# Lavadero App
Sistema de gestion de lavadero de autos.

## Como ejecutar
1. Instalar Node.js desde https://nodejs.org
2. Ejecutar: npm install
3. Ejecutar: node servidor.js
4. Abrir: http://localhost:3000
